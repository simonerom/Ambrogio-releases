Ambrogio

Hi, I'm Ambrogio.

I'm a household assistant, and I live in your house — on a Raspberry Pi on your own shelf. I read the family's email, keep the calendar, drive the lights, and remember what matters. Everything I know stays on that box: no account to create, no dashboard in someone else's data centre, no remote login left open. I think with Claude, on your own subscription. You flash a card, plug it in, and say hello.

Start here — write a card (macOS & Linux)
curl -LO https://github.com/simonerom/Ambrogio-releases/raw/main/tools/flash_box.py sudo python3 flash_box.py --flash

One little script downloads the latest image, flashes the card you pick, and writes your claim code. On Windows, or if you'd rather use a graphical tool, the flashing guide has the Raspberry Pi Imager route too.

Setting it up

Four steps, about twenty minutes.

You'll need a Raspberry Pi 4 or 5, a microSD card, and a computer to write it with — most of the twenty minutes is the card writing itself.

1

Run the script

The command above downloads the latest image, asks which removable disk is your SD card, and — after warning you — flashes it. One step, no separate imager. (On Windows or with a GUI, use Raspberry Pi Imager instead; the guide covers it.)

2

Print the code sheet

The same run writes a one-off claim code onto the card and hands you a sheet to print — the code, the box's Wi-Fi name, and a QR. It's your only copy, so print it and keep it. The code is what proves the box is yours when you claim it.

3

Boot the Pi, get it online

Put the card in, power it on, give it a few minutes. Then get it on your network: plug in an Ethernet cable, or let the app set up Wi-Fi for it (you'll scan the QR from your sheet).

4

Claim it from the app

The app finds the box on your network and asks you to claim it. Type your code to prove the box is yours, then give it a Claude subscription token so it can think — and say hello.

Why a code at all?

A fresh box announces itself on your network, and the first person to claim it becomes its owner — which means the family's email, an AI token, and an assistant that acts on your behalf. The code proves you're the one physically holding the box. Ambrogio never invents one for you: a code the box made up is a code nobody has. Lose the sheet and you re-inject it with the card in your hand — which is exactly the point.

The app

Set it up, then talk to it.

The app claims the box, walks you through setup, and is where you chat with Ambrogio day to day. There's a web console on the box too, for anything with a keyboard.

Android
Not yet

There's no Android build today, and we'd rather say so than show a button that doesn't work. The box's web console works in any browser on the same network, so Android isn't locked out of using Ambrogio — only out of the app.
